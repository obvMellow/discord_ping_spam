# discord_ping_spam
Stupid bot that spam pings someone

## Usage
- Create a `config.json` file that looks like this:
```json
{
    "guild_id": 123,
    "ping_id": 123,
    "token": "your bot token"
}
```
- Run the bot and hope that everything goes well
```sh
cargo run --release
```
- You should be getting the pings now
